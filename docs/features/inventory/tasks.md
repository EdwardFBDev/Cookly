# Inventory Feature Tasks

## Scope

Implement the connected Inventory, Ingredient Detail, Add Ingredient, Edit Ingredient, Expiring Ingredients, Location Management, and Inventory Empty screens.

## Tasks

1. Define inventory domain models and initial content.
2. Add local inventory state for list, detail, and add flows.
3. Build reusable inventory presentation components where they support the requested screens.
4. Implement Inventory screen.
5. Implement Ingredient Detail screen.
6. Implement Add Ingredient screen.
7. Add Expo Router routes for inventory list, detail, and add.
8. Connect existing Home navigation to the Inventory route.
9. Add edit ingredient update and delete behavior to local feature state.
10. Add expiring ingredients grouped review screen.
11. Add location management with local selection, creation placeholder, and delete relocation behavior.
12. Add inventory empty route and route active-location empty state into it.
13. Validate lint diagnostics for changed files.

## Out Of Scope

- SQLite repository implementation.
- Recipe matching engine.
- Image upload.
- Settings, shopping, and meal planning screens.
- Real location transfer progress from persistence.
